# A2UI Fixed Schema

## What This Demo Shows

Fixed-schema A2UI rendering where the UI schema is pre-defined in JSON files and only the data changes per invocation.

1. **Pre-built schemas**: Flight card layout loaded from `flight_schema.json`
2. **Data binding**: The agent populates flight data into the schema template
3. **Action handlers**: "Select" button triggers an optimistic booking confirmation
4. **No streaming**: All cards render at once after the tool completes
